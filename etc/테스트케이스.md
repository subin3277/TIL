# 테스트 케이스

### 테스트 베이시스(Test Basis)
* 정의 : 테스트 케이스를 만들기 위한 근거(데이터)
* 명세기반 베이시스
  - 문장으로 이루어진 내용
  - 기능설명서, 사용자 스토리
- 구조 기반 베이시스
  - 코드 제어 흐름도 등의 구조적 데이터로 이루어진 내용
  - 단위 코드, 제어 흐름도

### 테스트의 종류
- 사용자 요구명세서    - 인수 테스트
- 시스템 요구 명세서    - 시스템 테스트
- 시스템 설계    - 통합테스트
- 단위 설계    - 단위 테스트
- 코드

### 명세 기반 : 결정 테이블
- Condition을 가지고 상황별 테이블 만들기
- 결정 테이블(Decision Table) : 조건과 결과로 이루어진 논리 분석을 위한 표
